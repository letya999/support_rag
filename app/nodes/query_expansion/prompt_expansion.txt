- Create semantically diverse variations that capture the same intent from different angles
- Include synonyms, related terminology, and alternative phrasings
- Expand abbreviations and clarify ambiguous terms where appropriate

- Generate queries that would match different document styles (technical, conversational, formal)

Output format:
- Provide exactly 3 alternative queries
- Separate each query with a comma
- Output only the queries, no additional text or numbering
- Maintain the same language as the user's original question
